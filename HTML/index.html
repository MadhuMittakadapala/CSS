What is the CSS?

CSS stands for Cascading Style Sheet

Cascading stle sheet is the language we use for to style a webpages.CSS saves a lot of work,it can control the layout of multiple web pages all at once.

CSS handles the lok and fell part of a web pages, using css yu can control the colors of text and the style of fonts, the spacing between paragraphs , how columns are sized and laid out what background images or colors are used, layout designes,variations in display for differents devices and screen sizes.


CSS(Cascading Style Sheet) Advantages

1.CSS saves time:you can write css once and there use same sheet in multiple html pages, you can define a style for each html element and apply it to as many web pages as you want.

2.Pages load faster:if you are using css, you dont need to write html tag attributes every time,just write one cssrule of tag and apply it to all the occurances of that tag,so less code means faster download times.

3.Easy maintenance:to make a global change, simply change the style and all elements in all the web pages will be updated automatically.

4.Superior styles to HTML:CSS has a much wider arrays of the attributes than html,so you can give a far better look to html pages in comparison to html attributes.

5.Multiple device compatibility:style sheests allows contact to be optimized for more than one type of device.By using the same html document,different versions of website can be presented for handled devices such as PDAS and cellphones or printing.

6.Global web standards:now html attributes are being deprecated and it is being recommended to use css.so its a good idia to start using css. all the html pages to make them compatible to future browsers.




CSS syntax:

A CSS comprises of style rules that are interpreted by the browseer and them applied to the corresponding elements in document.

Selector { property : value }

example:

table { border : 1px solid #coo;}




CSS Selectors:

CSS selector selects the html elements you want to style.

we can divide css selectors into 5 categories:

1.Simple selectors (select elements based on name and id and class.)

2.Combinator selectors

3.Pseudo-class selectors

4.Pseudo-elements selectors

5.Attribute selectors



HOW TO ADD CSS TO HTML DOCUMENTS

when a browser reads a style sheet,it will format the HTML document according to the information in the style sheet.

there are 3 ways to insert css in html document :

1. Inline CSS

2. Internal CSS

3. External CSS


1.inline css: an inline css may be used to apply a unique style for a single element. To use inline css, add the style attribute to the relavent element. the style attributecan contain any css property.

2.internal css:an internal css style sheet may be used if one single html page has a unique style. The internal style is defined inside the 'style' element, inside the head section.

3.External css:with an external css you can change the look of an entire website by changing with just one file. Each html page must include a reference to the external style sheet file inside 'link'element, inside the head section.

an external style sheet can be written in any text editor,and must be saved with a .css extension.

the external css file should not contain any html tags.

NOTE:

DO NOT ADD A SPACE BETWEEN THE PROPERTY VALUE AND THE UNIT.

INCORRECT(SPACE)--> margin-left:20 px;

CORRECT(NO SPACE)--> margin-left: 20px;
